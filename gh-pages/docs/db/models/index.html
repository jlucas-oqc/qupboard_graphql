    ORM Models¶

    qupboard_graphql.db.models ¶

    SQLAlchemy ORM models mirroring the Pydantic HardwareModel schema.

    Table hierarchy¶

    hardware_models └─ qubits (FK → hardware_models) ├─ physical_channels (FK → qubits OR resonators, channel_kind = 'qubit' | 'resonator') │ baseband and IQ-bias columns inlined here ├─ pulse_channels (FK → qubits OR resonators, channel_role discriminator) │ channel_role values: │ 'drive' – qubit drive channel │ 'second_state' – second-state pulse channel (ss_active, ss_delay) │ 'freq_shift' – freq-shift pulse channel (fs_active, fs_amp, fs_phase) │ 'measure' – resonator measure channel │ 'acquire' – resonator acquire channel (acq_delay/width/sync/use_weights) │ 'reset_qubit' – qubit reset channel (reset_delay) │ 'reset_resonator' – resonator reset channel (reset_delay) │ └─ calibratable_pulses (owner_uuid → pulse_channels.id, pulse_role discriminator) ├─ cross_resonance_channels (FK → qubits, role = 'cr' | 'crc') │ └─ calibratable_pulses (pulse_role = 'cr') ├─ zx_pi_4_comps (FK → qubits, one per CR pair) │ └─ calibratable_pulses (pulse_role = 'zx_precomp' | 'zx_postcomp', nullable) │ phase_comp_x_pi_2 inlined on qubits └─ resonators (FK → qubits, one-to-one) ├─ physical_channels (FK → resonators, channel_kind = 'resonator') └─ pulse_channels (FK → resonators, channel_role = 'measure' | 'acquire' | 'reset_resonator')

    HardwareModelORM ¶

    Bases: RepositoryMixin, Base

    Top-level hardware model record containing version and qubit calibration data.

    Attributes:

    Name Type Description
    id Mapped[UUID]

    Auto-generated UUID primary key.
    version Mapped[str]

    Schema or calibration version string.
    calibration_id Mapped[str]

    Identifier for the calibration run.
    logical_connectivity Mapped[str]

    JSON-encoded mapping of qubit label to list of neighbour indices.
    qubits Mapped[list[QubitORM]]

    One-to-many collection of associated :class:QubitORM rows.

    PhysicalChannelORM ¶

    Bases: Base

    Unified physical channel row covering both qubit and resonator channels.

    A single table (physical_channels) stores physical channels for both qubits and resonators, discriminated by channel_kind. The inlined BaseBand and IQVoltageBias Pydantic fields are flattened into columns here rather than in separate tables.

    Attributes:

    Name Type Description
    uuid

    UUID primary key (shared with the Pydantic schema's UUID).
    channel_kind Mapped[str]

    "qubit" or "resonator" — identifies the owner type.
    name_index Mapped[int]

    Hardware name index used to address the physical port.
    block_size Mapped[int]

    Waveform block size in samples.
    default_amplitude Mapped[int]

    Default output amplitude in hardware units.
    switch_box Mapped[str]

    Switch-box identifier string.
    swap_readout_iq Mapped[bool]

    Whether to swap I and Q for readout (resonator only).
    baseband_uuid Mapped[UUID]

    UUID of the inlined baseband oscillator.
    baseband_frequency Mapped[float | None]

    Baseband carrier frequency in Hz.
    baseband_if_frequency Mapped[float | None]

    Intermediate frequency in Hz.
    iq_bias Mapped[str]

    Serialised IQ voltage bias correction string.
    qubit_uuid Mapped[UUID | None]

    FK to :class:QubitORM (set when channel_kind = "qubit").
    resonator_uuid Mapped[UUID | None]

    FK to :class:ResonatorORM (set when channel_kind = "resonator").
    qubit Mapped[QubitORM | None]

    Relationship to the owning :class:QubitORM.
    resonator Mapped[ResonatorORM | None]

    Relationship to the owning :class:ResonatorORM.

    CalibratablePulseORM ¶

    Bases: Base

    A single calibratable waveform pulse stored in the calibratable_pulses table.

    Pulses are owned by a channel row identified by owner_uuid and are further distinguished by the pulse_role discriminator column.

    Attributes:

    Name Type Description
    id Mapped[UUID]

    Auto-increment integer primary key.
    waveform_type Mapped[str]

    Waveform shape identifier (e.g. "gaussian").
    width Mapped[float]

    Pulse duration in seconds.
    amp Mapped[float]

    Pulse amplitude.
    phase Mapped[float]

    Pulse phase in radians.
    drag Mapped[float]

    DRAG correction coefficient.
    rise Mapped[float]

    Rise-time parameter (fraction of width).
    amp_setup Mapped[float]

    Setup amplitude.
    std_dev Mapped[float]

    Gaussian standard deviation.
    owner_uuid Mapped[UUID]

    UUID of the owning channel row.
    pulse_role Mapped[str]

    Discriminator string identifying which relationship slot this pulse fills (e.g. "drive", "drive_x_pi", "cr").

    PulseChannelORM ¶

    Bases: Base

    Unified pulse channel row covering all channel roles.

    A single table (pulse_channels) stores every role of pulse channel: drive, second_state, freq_shift, measure, acquire, reset_qubit, and reset_resonator. Role-specific columns are NULL for rows of a different role.

    Attributes:

    Name Type Description
    uuid

    UUID primary key.
    channel_role Mapped[str]

    Role discriminator ("drive", "second_state", "freq_shift", "measure", "acquire", "reset_qubit", "reset_resonator").
    frequency Mapped[float | None]

    Carrier frequency in Hz (None encodes NaN).
    imbalance Mapped[float]

    IQ imbalance correction factor.
    phase_iq_offset Mapped[float]

    IQ phase offset in radians.
    scale_real Mapped[float]

    Real part of the complex scale factor.
    scale_imag Mapped[float]

    Imaginary part of the complex scale factor.
    ss_active Mapped[bool | None]

    (second_state only) Whether second-state driving is active.
    ss_delay Mapped[float | None]

    (second_state only) Second-state delay in seconds.
    fs_active Mapped[bool | None]

    (freq_shift only) Whether frequency shifting is active.
    fs_amp Mapped[float | None]

    (freq_shift only) Frequency-shift amplitude.
    fs_phase Mapped[float | None]

    (freq_shift only) Frequency-shift phase.
    acq_delay Mapped[float | None]

    (acquire only) Acquisition delay in seconds.
    acq_width Mapped[float | None]

    (acquire only) Acquisition window width in seconds.
    acq_sync Mapped[bool | None]

    (acquire only) Whether to sync the acquisition.
    acq_use_weights Mapped[bool | None]

    (acquire only) Whether to use integration weights.
    reset_delay Mapped[float | None]

    (reset_qubit/reset_resonator only) Reset delay in seconds.
    qubit_uuid Mapped[UUID | None]

    FK to :class:QubitORM (set for qubit-owned channels).
    resonator_uuid Mapped[UUID | None]

    FK to :class:ResonatorORM (set for resonator channels).
    pulse Mapped[CalibratablePulseORM | None]

    Primary pulse waveform (most roles).
    pulse_x_pi Mapped[CalibratablePulseORM | None]

    X-π pulse waveform (drive role only).

    CrossResonanceChannelORM ¶

    Bases: Base

    Unified cross-resonance channel row covering both CR and CRC channels.

    A single table (cross_resonance_channels) stores both CrossResonancePulseChannel (role "cr") and CrossResonanceCancellationPulseChannel (role "crc") rows. The zx_pi_4_pulse relationship is only populated for "cr" rows.

    Attributes:

    Name Type Description
    uuid

    UUID primary key.
    role Mapped[str]

    "cr" for a cross-resonance channel or "crc" for a cross-resonance cancellation channel.
    auxiliary_qubit Mapped[int]

    Index of the auxiliary (target) qubit in the CR pair.
    frequency Mapped[float | None]

    Carrier frequency in Hz (None encodes NaN).
    imbalance Mapped[float]

    IQ imbalance correction factor.
    phase_iq_offset Mapped[float]

    IQ phase offset in radians.
    scale_real Mapped[float]

    Real part of the complex scale factor.
    scale_imag Mapped[float]

    Imaginary part of the complex scale factor.
    qubit_uuid Mapped[UUID | None]

    FK to the owning :class:QubitORM.
    qubit Mapped[QubitORM | None]

    Relationship to the owning :class:QubitORM.
    zx_pi_4_pulse Mapped[CalibratablePulseORM | None]

    ZX-π/4 calibration pulse (CR role only).

    ZxPi4CompORM ¶

    Bases: Base

    ZX-π/4 compensation element associated with a specific auxiliary qubit.

    Each row represents the calibration data for one CR-pair compensation, keyed by auxiliary_qubit index. Optional pre- and post-compensation pulses are stored as related :class:CalibratablePulseORM rows.

    Attributes:

    Name Type Description
    uuid

    Auto-generated UUID primary key.
    auxiliary_qubit Mapped[int]

    Index of the auxiliary (control) qubit in the CR pair.
    phase_comp_target_zx_pi_4 Mapped[float]

    Target ZX-π/4 phase compensation.
    pulse_zx_pi_4_target_rotary_amp Mapped[float | None]

    Optional rotary pulse amplitude.
    precomp_active Mapped[bool]

    Whether pre-compensation is enabled.
    postcomp_active Mapped[bool]

    Whether post-compensation is enabled.
    use_second_state Mapped[bool]

    Whether to use the second excited state.
    use_rotary Mapped[bool]

    Whether to use a rotary pulse.
    qubit_uuid Mapped[UUID | None]

    FK to the owning :class:QubitORM.
    qubit Mapped[QubitORM | None]

    Relationship to the owning :class:QubitORM.
    pulse_precomp Mapped[CalibratablePulseORM | None]

    Optional pre-compensation :class:CalibratablePulseORM.
    pulse_postcomp Mapped[CalibratablePulseORM | None]

    Optional post-compensation :class:CalibratablePulseORM.

    ResonatorORM ¶

    Bases: Base

    Readout resonator coupled one-to-one with a :class:QubitORM.

    Holds the resonator's physical channel and its three pulse channels (measure, acquire, reset_resonator).

    Attributes:

    Name Type Description
    uuid

    UUID primary key (shared with the Pydantic schema's UUID).
    qubit_uuid Mapped[UUID]

    FK to the parent :class:QubitORM.
    qubit Mapped[QubitORM]

    Relationship to the parent :class:QubitORM.
    physical_channel Mapped[PhysicalChannelORM | None]

    Single :class:PhysicalChannelORM with channel_kind = "resonator".
    pulse_channels Mapped[list[PulseChannelORM]]

    All resonator pulse channels (measure, acquire, reset_resonator).

    measure_channel property ¶

    Return the measure pulse channel, or None if absent.

    acquire_channel property ¶

    Return the acquire pulse channel, or None if absent.

    reset_resonator_channel property ¶

    Return the resonator reset pulse channel, or None if absent.

    QubitORM ¶

    Bases: Base

    ORM representation of a single qubit and its full calibration payload.

    Owns a physical channel, multiple pulse channels (drive, second_state, freq_shift, reset_qubit), cross-resonance channels, ZX-π/4 compensation entries, and a one-to-one resonator.

    Attributes:

    Name Type Description
    uuid

    UUID primary key.
    qubit_key Mapped[str]

    String key used to identify the qubit within the model (e.g. "q0").
    mean_z_map_args Mapped[str]

    JSON-encoded two-element list [real, imag] representing the mean-Z mapping arguments.
    discriminator_real Mapped[float]

    Real part of the state-discrimination threshold.
    discriminator_imag Mapped[float]

    Imaginary part of the state-discrimination threshold.
    direct_x_pi Mapped[bool]

    Whether to use a direct X-π pulse.
    phase_comp_x_pi_2 Mapped[float]

    X-π/2 phase compensation value.
    hardware_model_id Mapped[UUID | None]

    FK to the parent :class:HardwareModelORM.
    hardware_model Mapped[HardwareModelORM | None]

    Relationship to the parent :class:HardwareModelORM.
    physical_channel Mapped[PhysicalChannelORM | None]

    Single :class:PhysicalChannelORM with channel_kind = "qubit".
    pulse_channels Mapped[list[PulseChannelORM]]

    All qubit-owned pulse channels.
    resonator Mapped[ResonatorORM | None]

    One-to-one :class:ResonatorORM.
    cross_resonance_channels Mapped[list[CrossResonanceChannelORM]]

    CR pulse channels (role "cr").
    cross_resonance_cancellation_channels Mapped[list[CrossResonanceChannelORM]]

    CRC pulse channels (role "crc").
    zx_pi_4_comps Mapped[list[ZxPi4CompORM]]

    ZX-π/4 compensation rows.

    drive_channel property ¶

    Return the drive pulse channel, or None if absent.

    second_state_channel property ¶

    Return the second-state pulse channel, or None if absent.

    freq_shift_channel property ¶

    Return the frequency-shift pulse channel, or None if absent.

    reset_qubit_channel property ¶

    Return the qubit reset pulse channel, or None if absent.
